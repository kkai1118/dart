# 24 补充：10 个高频例子

对应主文档章节：`24. 10 个高频例子`

## 怎么使用这些例子

不要只看，建议每个例子都自己改两次：

- 改变量名
- 改类型
- 改返回值
- 改成错误写法再看报错

## 一个示范

原例子：

```dart
int add(int a, int b) => a + b;
```

你可以改成：

```dart
double add(double a, double b) => a + b;
```

再改成：

```dart
String add(String a, String b) => a + b;
```

这样你会更快理解“类型”和“函数签名”的关系。
