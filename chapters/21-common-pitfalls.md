# 21 补充：初学者常见坑

对应主文档章节：`21. 初学者常见坑`

## 坑 1：把 `var` 当成弱类型

`var` 只是省略类型写法，不是“随便换类型”。

## 坑 2：看到 `?`、`!` 就慌

把它们拆开理解：

- `?`：允许为空
- `!`：强制告诉编译器这里不为空

## 坑 3：分不清“变量不可变”和“对象不可变”

```dart
final list = [1, 2];
list.add(3); // 合法
```

## 坑 4：函数调用和对象创建看混

```dart
User(name: 'Tom')
```

这是构造函数调用，也是在创建对象。

## 坑 5：见到长 Flutter 代码就不敢拆

其实通常就是：

- 创建对象
- 传命名参数
- 参数里再创建子对象
